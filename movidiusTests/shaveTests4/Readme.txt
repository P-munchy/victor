HelloWorldShave

Supported Platform
===================
Depending on the platform chosen:
Myriad1 - This example works on Myriad1 silicon and simulator.             
Myriad2 - This example works on Myriad2 simulator only. 

Overview
==========
Just print hello world using shave compiled code using stdio library

Hardware needed
==================
Depending on the platform chosen:
Myriad1 - This software runs on MV153 board.
Myriad2 - No available hardware for this platform yet.

Build
==================
Please type "make help" if you want to learn available targets.

To build the project please type:
"make clean"
"make all"


Setup
==================
Depending on the platform chosen:
Myriad1 - To run the application:
    - open terminal and type "make start_server"
    - open another terminal and type "make run"
Myriad2 - To run the application:
    - open terminal and type "make start_simulator"
    - open another terminal and type "make run"


Expected output
==================
 The result consist in printf seen using the debugger.
 Hello from SHAVE!
